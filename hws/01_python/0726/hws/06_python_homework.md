## 1. Built-in 함수와 메서드

```
sorted()는 그 자체는 정렬되지 않고 정렬된 값을 반환하는데 반해
sort()는 그 자체가 정렬되고 None을 반환한다.
이를
a = [5,3,1,2,4]
print(a.sorted())   [1,2,3,4,5]
print(a)            [5,3,1,2,4]
print(a.sort())     None
print(a)            [1,2,3,4,5]
를 통해 알 수 있다.
```

## 2. .extend()와 .append()

```
이 둘은 문자열을 넣을때 제일 큰 차이점을 보인다.
a = ['1', '2']
a.append('345')
print(a)            ['1','2','345']
a.extend('678')
print(a)            ['1','2','345','6','7','8']

```

(모범답안)
extend() 메서드는 인자로 들어오는 iterable한 객체를 순회하며 요소를 하나씩 리스트에 삽입한다.
